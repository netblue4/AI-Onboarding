3. Data Resources

Field Description Example
Dataset ID Unique identifier for this dataset within the project. DS 001
Dataset Name Common or descriptive name of the dataset. Customer Churn Data Q1 2023-Q4 2024
Description Brief overview of the dataset's content, origin, and general purpose. Historical data of customer subscriptions, usage, and churn status.
Source Where the data originates from (e.g., internal database, third-party vendor, public repository, synthetic generation). Internal CRM Database (Salesforce Extract)
Intended Use Specific purpose(s) in the AI lifecycle (e.g., training, validation, testing, fine-tuning, specific model). Training primary churn prediction model, testing fairness metrics.
Data Type(s) Main types of data contained (e.g., numerical, categorical, text, image, time-series). Numerical, Categorical, Datetime
Format File format or storage system (e.g., CSV, JSON, Parquet, SQL DB, ImageNet format). CSV, Parquet
Approximate Size Estimated size (e.g., number of records, GB, TB). ~5M records, ~2GB
Access Method/Location How the data can be accessed (e.g., S3 bucket path, API endpoint, database query, shared drive). S3://project-bucket/raw_data/churn/
Owner/Custodian Person or team responsible for the data. Sales Operations Team
Date Identified Date the dataset was identified as a candidate. 2025-05-29
Licensing/Usage Rights Any licensing terms, usage restrictions, or consent requirements. Internal use only, GDPR compliant consent obtained.
Sensitivity/PII Indicate if Personally Identifiable Information (PII) or other sensitive data is present and its nature. Contains PII (Names, Email), Anonymization required for modeling.
